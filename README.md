### Advent of Code 2020

This repo contains the code I'm using to solve [Advent of Code 2020](https://adventofcode.com/2020).

### How to run the code

```
❯ go build
❯ ./advent-of-code-2020 --help
Usage of ./advent-of-code-2020:
  -day int
        the day's code to target for execution
  -part int
        1 or 2 (default 1)

```
